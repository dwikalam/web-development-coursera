HTML Characters Entity References

There are three characters that should always be escaped so they do not cause any rendering issues (between be treated as HTML and content):

  • Greater than sign (>): use "&gt;" instead of ">"
  • Less than sign (<): use "&lt;" instead of "<"
  • Ampersand sign (&): use "&amp;" instead of "&"
